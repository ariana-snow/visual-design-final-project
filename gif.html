GIF

I created this GIF as a promotional image for a social media campaign for my Cupcake Town website. The image will be used on Twitter and Facebook as an eye-catching marketing feature to attract more people to the website. The GIF was created and animated in GIMP. I chose the size to be 600 px by 600 px for optimal web resolution. I uploaded the principal icon for my cupcake website and I created five layers. I changed the color of each cupcake layer before animating it and exporting it as a gif image, with a never-ending loop cycle.
